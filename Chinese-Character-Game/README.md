# 小小汉字勇士

这是一个零依赖的中文识字闯关游戏原型，目标用户是儿童。

## 当前版本包含

- 125 关课程结构
- 每关 20 个汉字，共 2500 个常用汉字
- 本关 20 字一览，可以快速预览并点击查看大字卡
- 先预览汉字，再点击“开始闯关”进入测试
- 本关进度条显示已会字数
- 每个汉字都要答对一次才算会认
- 答错的字会自动进入错题本，并在队伍后面再次出现
- 本关 20 个字全部会认后，解锁小游戏选择：汉字泡泡、接宝箱或汉字贪食蛇
- 汉字泡泡显示大号拼音目标，需要在泡泡飘走前点掉对应汉字
- 接宝箱显示大号拼音目标，需要左右移动篮子接住正确汉字，接错字或炸弹会消耗生命
- 贪食蛇显示大号拼音目标，需要自己记住对应汉字
- 贪食蛇默认困难模式，撞墙或吃错会消耗生命；可切换轻松模式穿墙
- 任意小游戏完成后获得星星和贴纸，并解锁下一关；每关完成后可再玩，最多 3 次
- 错题本可生成 A4 打印页，可在浏览器中另存为 PDF
- 本地进度保存，关闭页面后不会丢失

## 怎么运行

1. 直接双击打开 `index.html`
2. 或者在浏览器中打开：

```text
file:///C:/Users/leene/Documents/Chinese-Character-Game/index.html
```

## 课程数据

- `3500-common.txt` 是常用字来源
- `hanzi-pinyin-table.json` 是拼音来源
- `tools/generate-word-data.js` 会生成 `word-data.js`
- 当前 `word-data.js` 已生成 125 x 20 的正式关卡结构

重新生成课程数据：

```bash
node tools/generate-word-data.js
```
